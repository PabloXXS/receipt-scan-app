# Вклад

## Требования

- Flutter >= 3.22, Dart >= 3.4
- Линтеры: `flutter_lints` (см. `analysis_options.yaml`)
- Форматирование: `dart format`

## Команды

```bash
flutter pub get
flutter analyze
flutter test
```

## GitHub Actions

- CI: сборка, анализ, тесты
- Секреты: `SUPABASE_URL`, `SUPABASE_ANON_KEY`, `SENTRY_DSN`
